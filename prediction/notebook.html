def normalize_name(name):
    alphabet = set(string.ascii_lowercase)
    name = str(name).lower()
    new_name = ''
    for l in name:
        if l == ' ':
            new_name += '_'
        elif l in alphabet:
            new_name += l
    
    return new_name

def process_data(raw_df, drop_odds = True):
    length = raw_df.shape[0]
    year1 = np.zeros(length, dtype=int)
    year2 = np.zeros(length, dtype=int)
    date_day = np.zeros(length, dtype=int)
    date_month = np.zeros(length, dtype=int)
    date_year = np.zeros(length, dtype=int)
    
    for i, row in raw_df.iterrows():
        y1, y2 = row['year'].split('-')
        year1[i] = int(y1)
        year2[i] = int(y2)
        d, m, y = row['Date'].split('/')
        date_day[i] = int(d)
        date_month[i] = int(m)
        date_year[i] = int(y)
        raw_df.at[i,'AwayTeam'] = normalize_name(row['AwayTeam'])
        raw_df.at[i,'HomeTeam'] = normalize_name(row['HomeTeam'])
        raw_df.at[i,'Referee'] = normalize_name(row['Referee'])
        
    
    # add new features
    raw_df['year1'] = pd.Series(year1, index=raw_df.index)
    raw_df['year2'] = pd.Series(year2, index=raw_df.index)
    raw_df['date_day'] = pd.Series(date_day, index=raw_df.index)
    raw_df['date_month'] = pd.Series(date_month, index=raw_df.index)
    raw_df['date_year'] = pd.Series(date_year, index=raw_df.index)
    
    away_team = pd.get_dummies(raw_df['AwayTeam'], prefix='AwayTeam')
    home_team = pd.get_dummies(raw_df['HomeTeam'], prefix='HomeTeam')
    referee = pd.get_dummies(raw_df['Referee'], prefix='Referee')
    
    raw_df = pd.concat([raw_df, away_team, home_team, referee], axis=1)
    
    # remove old features
    raw_df = raw_df.drop(['year', 'Div', 'Date', 'AwayTeam', 'HomeTeam', 'Referee'], 
                         axis=1, errors='ignore')
    
    # deal with missing values
    fill_NaN = Imputer(missing_values=np.nan, strategy='mean', axis=1)
    imputed_DF = pd.DataFrame(fill_NaN.fit_transform(raw_df))
    imputed_DF.columns = raw_df.columns
    imputed_DF.index = raw_df.index
    
    if drop_odds:
        imputed_DF = imputed_DF.drop(['BSA', 'BSD', 'BSH', 'BWA', 'BWD', 'BWH', 'Bb1X2', 
                                      'BbAH', 'BbAHh', 'BbAv_lt_2.5', 'BbAv_bt_2.5', 'BbAvA', 
                                      'BbAvAHA', 'BbAvAHH', 'BbAvD', 'BbAvH', 'BbMx_lt_2.5', 
                                      'BbMx_bt_2.5', 'BbMxA', 'BbMxAHA', 'BbMxAHH', 'BbMxD', 
                                      'BbMxH', 'BbOU', 'GBA', 'GBD', 'GBH', 'IWA', 'IWD', 'IWH', 
                                      'LBA', 'LBD', 'LBH', 'PSA', 'PSD', 'PSH', 'SBA', 'SBD', 'SBH', 
                                      'SJA', 'SJD', 'SJH', 'VCA', 'VCD', 'VCH', 'WHA', 'WHD', 'WHH'], 
                                     axis=1, errors='ignore')
        # drop necessary odds
    
    
    return imputed_DF

Investigate relationship between Bets and Results¶

While a subset of bookmakers appear to give the same odds, we cannot draw any generalized insights by simply looking at the raw odds.

Essentially, betting odds are public opinions of match outcomes, with participants voting by their cash. As the result, we can calculate the implied probabilities of each game given the odds. Let's see whether we can discover any trend from the implied probabilities.

Implied Probability given Odds: ¶

Notice, different combinations of odds may produce the same implied probabilities. For example, let's say booker A offers 2.0 for Home, Away, and Draw, indicating implied probabilities of 1/3 for each outcome. Booker B offers 3.0 for each of the three outcomes, also ending with implied probabilities of 1/3.

Because the magnitude of raw odds varies, it is quite difficult to visualize them. Therefore, we decide to look into implied probabilities, since they are essentially normalized odds.

# Extract all inplied probabilities from the table
# 1. group by the same companies
# 2. compute the tuple of probabilities
# 3. return the new table

# input: row = [A,D,H, A,D,H, A,D,H,...] in odds form
# output row in normalized probability form
def implied_prob_row(row):
    assert(len(row) % 3 == 0)
    prob = []
    idx = row.index.map(lambda x: x[:-1]).unique()
    for i in idx:
        prob_temp = []
        cur_odds = row[[i+'A', i+'D', i+'H']]
        if close(sum(cur_odds), 1):
            print('Already Normalized')
            return row
        if cur_odds[0] == np.nan:
            prob += cur_odds
            continue
        for o in cur_odds:
            prob_temp.append(1.0/o)
        
        prob += (normalize(prob_temp))
    return pd.Series(prob, index=row.index)

def normalize(L):
    tot = sum(L)
    return [i/tot for i in L]

def close(x,y):
    return abs(x-y) < 0.000000001

Can we design some strategies to profit off the predictions?¶

Strategies ¶

* Bet along bookmakers' predictions
* Bet against bookmakers' predictions
* Bet according to the Logistic Regression model

How would our return look like?

Notice, we obtain different returns by betting with different bookers. If we bet with booker A, in order to get 100 dollars back regardless of any outcome, we need to place 50 for each of the actions, meaning we need to pay 150 to guarantee a 100 return. On the other hand, by doing business with booker B, we are guaranteed a 100 return by betting 33.33 in all three actions.

As the result, choosing a good bookie is significant to our expected return!

For this analysis, we decided to use B365 data for three reasons:

1. B365 is among the three bookmakers that have no missing data.
2. B365 consistently has the highest odds among its peers.
3. Our model is built with B365 features.

        if row[i+'H'] == np.nan:
            returns.append(np.nan)
        elif cur_pred == row['FTR']:
            returns.append(bet_size * cur_odds[i+cur_pred])
        else:
            returns.append(0)

    return pd.Series(returns, index = return_idx)
